## Consigna

- Explica la diferencia entre un Parámetro y un Argumento. 

## Respuesta

El parámetro es una variable que se declara dentro de un argumento y que si no tiene algún “out”no se puede llegar a utilizar fuera del código, se inicia ahí y se queda ahí. Mientras que un Argumento es una variable que se inicia fuera del código y que nunca se borra a menos de que lo explicites, también funciona como una referencia para el valor de variables dentro de un Método.
