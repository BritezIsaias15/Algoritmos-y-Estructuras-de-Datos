## Consigna

- ¿En qué situación preferirías usar un parámetro out en lugar de un ref?

## Respuesta

Preferiría utilizar un out si llegara a necesitar que múltiples variables me den valores y luego pasarlas al código principal, en vez de usar un ref dónde tendría que tomarme la tarea de crear las variables previamente en el main y luego modificarlas.
