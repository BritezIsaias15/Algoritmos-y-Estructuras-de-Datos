## Consigna

- Si paso un **int x = 10** a un método **Duplicar(int n)** por valor, ¿qué valor tendrá **x** después de la llamada si adentro del método hice
    **n = n * 2**? 

## Respuesta

El valor de x seguirá siendo 10 porque el método toma la x como un valor para definir a n sin la posibilidad de regresarlo ni cambiar su valor actual.
