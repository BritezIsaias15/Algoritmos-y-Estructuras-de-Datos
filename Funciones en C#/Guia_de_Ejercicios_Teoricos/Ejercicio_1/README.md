## Consigna

- ¿Cuál es la diferencia principal entre un método void y uno que devuelve un int? 2. ¿Qué sucede con el código de un método que se encuentra escrito debajo de una instrucción return? 

## Respuesta

La diferencia entre un método void y uno que devuelve un int es que la función void solo ejecuta sus líneas de código sin necesidad de devolver un valor, mientras que con el método con retorno ejecuta sus líneas y necesita obligatoriamente una línea de “return();”.
El código debajo de un return es considerado inalcanzable por parte del compilador.
