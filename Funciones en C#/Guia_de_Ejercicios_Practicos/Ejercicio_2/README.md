## Consigna

Diseñar un sistema de conversión que utilice dos funciones distintas:

- **CelsiusAFahrenheit:** Recibe un double (grados celsius) y devuelve el equivalente en fahrenheit.

- **FahrenheitACelsius:** Recibe un double (grados fahrenheit) y devuelve el equivalente en celsius.

- **Fórmula:** Fahrenheit = (Celsius * 9/5) + 32

- **Desafío:** Crear un menú en el Main que permita al usuario elegir qué conversión realizar.
