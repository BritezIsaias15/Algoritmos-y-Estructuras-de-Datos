# Consigna

¿Qué son las dimensiones de una matriz? Si tienes una matriz A con m filas y n columnas, ¿cómo se denota su tamaño?

# Desarrollo

Las dimensiones de una matriz son el número de filas y columnas que posee. Se denotaría como mxn, siendo m fila horizontales (de izquierda a derecha) y n columnas verticales (de arriba a abajo).
