# Consigna

Si la matriz A es de tamaño m×n y la matriz B es de tamaño p×q, ¿cuáles deben ser los valores de n y p para que la multiplicación sea posible? ¿Cuál será el tamaño de la matriz resultante C?

# Desarrollo

Los valores de n y p deberían ser iguales, por ejemplo, n = 2 y p = 2. El tamaño de la matriz C Sería m = filas del A, n = columnas del B.
