# Consigna

Explica conceptualmente cómo se calcula un elemento C[i][j] de la matriz producto.

# Desarrollo

Se obtiene mediante el producto entre la fila i de A y la columna j de B. Se multiplica el primer valor de la fila i con el primer valor de la columna j, luego el segundo con el segundo y así hasta finalizar. El resultado sumado de esos valores da como resultado el valor de la primera posición donde empezó la multiplicación.
