# Consigna

Si tienes dos matrices A y B del mismo tamaño, ¿cómo se realiza la operación de suma A + B? Describe el proceso a nivel de elementos.

# Desarrollo

Para sumarlas hay que sumar el valor de una matriz con el otro valor de la matriz de la misma posición.
A=(1, 3),    B=(8, 4),
  (4, 2)       (6, 2)

Suma = ((1+8), (3+4)), = (9,  7),
       ((4+6), (2+2))    (10, 4)
