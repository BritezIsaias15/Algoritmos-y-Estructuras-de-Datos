# Diagnóstico y Logs (log4net) 

## Consigna

¿Qué información crítica aporta el Stack Trace al desarrollador en un entorno de producción y cómo ayuda log4net a persistir esta información? 

## Respuesta

El Stack Trace ayuda a encontrar el punto de origen de la excepción, muestra la ruta tomada por el código antes de la excepción. Log4Net captura la ruta del error para luego ser visto por el programador.
