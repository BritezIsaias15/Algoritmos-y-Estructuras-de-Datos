#  Caso Base vs. Caso Recursivo

## Consigna

En el cálculo de un Factorial o la serie de Fibonacci, ¿qué sucedería si el Caso Base está mal definido o es inalcanzable? 

## Respuesta

Si el Caso Base fuera inalcanzable el código se repetirá infinitamente hasta que sobrepase el límite del call stack y se produzca el error StackOverflowException.
