 # Eliminación de Duplicados en un Arreglo Ordenado

Dado un arreglo de enteros que ya se encuentra ordenado de menor a mayor de forma ascendente, escribe un algoritmo que elimine todos los elementos duplicados directamente en ese arreglo, de modo que cada elemento aparezca solo una vez. Los elementos válidos deben quedar al principio del arreglo y la función debe retornar la nueva longitud lógica del arreglo.

 - **Concepto clave:** Técnica de dos punteros (lectura y escritura) sobre estructuras contiguas.
