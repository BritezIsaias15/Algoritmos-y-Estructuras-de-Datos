 # Rotación de Elementos

Desarrolla una función que reciba un arreglo de enteros y un número de posiciones a rotar. El programa debe desplazar los elementos del arreglo hacia la derecha esa cantidad de veces. Por ejemplo, si el arreglo es [1, 2, 3, 4, 5] y se rota 2 posiciones, el resultado debe ser [4, 5, 1, 2, 3]. Optimiza el código para que funcione eficientemente incluso si el número de rotaciones es mayor que la longitud del arreglo.

 - **Concepto clave:** Operador residuo (%) y algoritmos de rotación eficiente.
