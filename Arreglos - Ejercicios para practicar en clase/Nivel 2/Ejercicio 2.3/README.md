# Frecuencia de Elementos

Escribe un programa que reciba un arreglo de 10 caracteres e imprima en consola cuántas veces se repite cada carácter. (Por ejemplo: si el arreglo tiene ['a', 'b', 'a', 'c'], debe indicar que 'a' aparece 2 veces, 'b' 1 vez, etc.).

 - **Concepto clave:** Control de elementos ya visitados o uso de arreglos auxiliares de conteo.
