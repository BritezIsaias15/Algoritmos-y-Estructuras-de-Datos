# Filtrar y Copiar (Números Primos)

Crea un arreglo original con 15 números aleatorios entre 1 y 50. Luego, identifica cuántos de ellos son números primos y transpórtalos a un nuevo arreglo que tenga el tamaño exacto para contenerlos (sin dejar espacios vacíos ni ceros al final).

 - **Concepto clave:** Arreglos dinámicos simulados, funciones de verificación y dimensionamiento preciso.
