# Consigna 
Escribir una función/método llamado productoDeArreglos que toma 2 arreglos de enteros del mismo tamaño y devuelve un nuevo arreglo del mismo tamaño que el de los de entrada. Si llamamos a al primer arreglo de entrada y b al segundo arreglo de entrada, los elementos del arreglo resultado deben ser generados por la siguiente fórmula: c[i] = a[i] * b[N – (i + 1)], donde llamamos N al tamaño de los arreglos.
