# Consigna

Escribir una función/método llamado busquedaLineal que toma un arreglo de enteros y un entero. Si el entero se encuentra entre los elementos del arreglo, el método debe devolver la posición en la que se encuentra, caso contrario debe devolver un valor que permita identificar a quien llama a la función/método que el entero no se encuentra en el arreglo.
