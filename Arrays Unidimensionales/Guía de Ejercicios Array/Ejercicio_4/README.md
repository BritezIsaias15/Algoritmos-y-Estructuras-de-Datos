# Consigna

Escribir una función/método llamado busquedaB que toma un arreglo de enteros ordenados de manera ascendente y un entero. El método debe buscar el entero en el arreglo de manera eficiente, aprovechando que los elementos del arreglo se encuentran ordenados de manera ascendente. Si el entero se encuentra entre los elementos del arreglo, el método debe devolver la posición en la que se encuentra, caso contrario debe devolver un valor que permita identificar a quien llama a la función/método que el entero no se encuentra en el arreglo.
