# Consigna

Escribir una función/método llamado estrictamenteCreciente que toma un arreglo de enteros y verifica que los elementos de este se encuentren en orden estrictamente creciente. El método debe devolver la cantidad de veces que se rompe el orden estrictamente creciente de los elementos.

Ejemplo: { 1, 3, 5, 2, 4, 6, 8, 8, 9, 10}, el número de veces que se rompe el orden es 2.
