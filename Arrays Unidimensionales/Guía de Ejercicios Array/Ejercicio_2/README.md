# Consigna

Escribir una función/método llamado busquedaL que toma un arreglo de enteros y unentero. Si el entero se encuentra entre los elementos del arreglo, el método debe devolverla posición en la que se encuentra, caso contrario debe devolver un valor que permita identificar a quien llama a la función/método que el entero no se encuentra en el arreglo.
