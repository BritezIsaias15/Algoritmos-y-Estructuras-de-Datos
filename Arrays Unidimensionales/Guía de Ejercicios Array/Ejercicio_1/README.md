# Consigna

Escribir una función/método llamado productoEscalarConInverso que toma 2 arreglos de enteros del mismo tamaño y devuelve el producto escalar entre el primer arreglo y el arreglo que resulta de los elementos del segundo arreglo en orden inverso.

**Nota:**  no debería ser necesario duplicar el código para calcular el producto escalar ni el
código necesario para invertir los elementos de un arreglo. En ambos casos se debería
poder utilizar los métodos creados para la resolución de ejercicios anteriores.
